Чтобы открыть пг админ с базой нужно зайти в cmd и прописать: 

- docker ps 
найти там контейнер с бд, и скопировать хеш

- docker inspect <hash>
найти поле "IPAddress" и это и есть локальный адрес базы

собственно и войти по этому адресу.
